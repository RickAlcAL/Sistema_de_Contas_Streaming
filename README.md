# 📺 Sistema de Contas Streaming

## 📝 Descrição
Console application em Java estruturada para o gerenciamento completo de uma plataforma de streaming. O sistema simula o controle de assinaturas, perfis de usuários e o catálogo de mídia, priorizando a organização de código através de herança e proteção de dados.

## 🧱 Pilares de POO Aplicados
* **Encapsulamento:** Controle rígido sobre o status da assinatura e dados de login do usuário, impedindo alterações externas diretas na conta.
* **Herança:** Estruturação de categorias de contas (ex: `PlanoBasico`, `PlanoPremium`) herdando de uma classe mãe `Assinatura`, além da divisão de mídias (`Filme`, `Serie`) herdando de `Midia`.

## 🚀 Funcionalidades
* Criação e gerenciamento de perfis com controle de acesso.
* Controle de status da assinatura (Ativa/Inativa).
* Catálogo de conteúdos interativo.

## 🛠️ Tecnologias Utilizadas
* **Java SE**
* **Paradigma:** Orientação a Objetos (POO)

## 💻 Como Executar
1. Clone o repositório.
2. Navegue até a pasta do projeto.
3. Abra em sua IDE de preferência.
4. Execute a classe principal (`Main.java`).
