# Sistema_de_Contas_Streaming
Sistema em Java desenvolvido para gerenciar plataformas de streaming de vídeo. Aplica herança para categorizar planos de assinatura e mídias (filmes e séries), além de encapsulamento para controle de perfis e dados de acesso. Permite gerenciar o status de contas e navegar pelo catálogo de maneira organizada.
